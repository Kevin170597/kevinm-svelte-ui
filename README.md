## Installation

Install SvelteKev with npm

```bash
  npm install sveltekev
```
Tailwind is required!
    
## Usage/Example

```javascript
import { Button } from 'sveltekev'

<Button 
    onClick={() => alert('hello')}
>
    Alert!
</Button>
```
